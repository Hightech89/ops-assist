# AI Development Guide

## General Rules

- Keep implementations simple
- Avoid unnecessary abstractions
- Do not overengineer
- Maintain TypeScript strictness
- Preserve consistent UI styling
- Keep components readable

## UI Direction

Industrial modern UI:
- dark slate backgrounds
- subtle amber/orange accents
- clean card layouts
- responsive design
- operator-friendly readability

## Preferred Stack

- Next.js App Router
- TypeScript
- Tailwind CSS

## Development Priorities

1. Clean UI
2. Reliable functionality
3. Good UX
4. Clear code structure

## Avoid

- Premature optimization
- Complex state management
- Large dependency additions
- Fancy animations
- Unnecessary backend complexity
