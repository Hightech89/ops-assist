# Ops Assist

AI-powered troubleshooting assistant for factory operators built with Gemma 4.

## Overview

Ops Assist helps manufacturing operators troubleshoot machine issues using AI-powered reasoning and contextual analysis.

The project is being built for the DEV.to Gemma 4 Challenge.

## Goals

- Help operators diagnose machine issues faster
- Provide beginner-friendly troubleshooting guidance
- Demonstrate practical AI use in manufacturing environments
- Explore local/open AI models for industrial workflows

## Planned Features

- Machine issue analysis
- Troubleshooting recommendations
- Safety reminders
- Escalation guidance
- Plain-English explanations

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Gemma 4
