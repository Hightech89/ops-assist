# MVP Scope

## Included In MVP

- Single-page application
- Industrial-themed UI
- Input form for troubleshooting issues
- Mock AI response initially
- Later integration with Gemma 4 API
- Responsive layout
- Basic troubleshooting output cards

## NOT Included In MVP

- Authentication
- Multi-user support
- Database persistence
- Real factory integrations
- Image upload support
- Admin dashboards
- Analytics
- Mobile apps
- Voice input
- Complex workflows

## Success Criteria

The MVP is successful if:
- A user can describe a machine issue
- The app returns realistic troubleshooting guidance
- The UI looks polished enough for screenshots/demo
- Gemma 4 integration works reliably
